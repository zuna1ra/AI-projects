# Advanced Computer Vision Project
### VGG • ResNet • YOLO • Object Tracking

**Author:** Zunaira  
**Field:** Artificial Intelligence / Computer Vision  

---

## Project Overview

This project demonstrates advanced computer vision techniques using deep learning models:

- Image classification using **VGG**
- Image classification using **ResNet**
- Real-time object detection using **YOLO**
- Object tracking across frames
- Performance analysis and visualization

The goal is to understand modern deep learning architectures and their real-world applications in visual recognition and tracking.
