**NLP Text Classification

Overview**
This project performs text classification using NLP techniques such as preprocessing, Bag of Words, and TF-IDF.

**Features**
- Text preprocessing
- BoW feature extraction
- TF-IDF feature extraction
- Model training and evaluation

**Technologies Used**
- Python
- Pandas
- Scikit-learn
- NLTK

**How to Run**
Open the notebook in Jupyter or Google Colab and run all cells.
